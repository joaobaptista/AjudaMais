Projeto Olhar+

O Projeto Olhar+ busca valorizar o seu empenho em ajudar quem precisa, retribuindo com Wallpapers exclusivos e fotografias impressas assinadas e com tiragem limitada (para receber a fotografia impressa, é necessário informar um endereço em território nacional. No momento não estamos enviando para o exterior).

Como funciona?
Basta efetuar qualquer doação de R$10.00 ou mais, seja para o projeto "Máscaras para Todos", ou para qualquer outro projeto social/hospital/ONG cadastrada no portal AJUDA+.

A cada R$10.00 em doações, você terá direito a 1 (UM) Wallpaper para celular ou computador. Para doações de valor maior ou igual a R$50.00, você poderá escolher entre uma foto impressa em papel Satin fotográfico, no tamanho A4 ou 5+ Wallpapers. 100% dos valores arrecadados pelo projeto Olhar+ ao "Máscara para Todos" será destinado à confecção de máscaras e compra de produtos de higiene para ser distribuídos para pessoas em situação de rua, projetos sociais, entregadores e motoristas de aplicativos. O valor do envio das fotos, via Correios, será pago pelos fundadores da plataforma.

Para doar
Residente no Brasil: Vakinha Online ou, para doações de até R$24.99, no ABACA$HI
Residente no Exterior: GofundMe
Se preferir, solicite nossos dados bancários pelo formulário abaixo ou pelo email contato@ajudamais.org

Todos os valores arrecadados, bem como as doações poderão ser visualizadas no LINK

Solicitando Wallpapers:
Preencha o formulário e anexe o comprovante da sua doação, que enviaremos o link para download em até 24 horas: Clique Aqui para acessar o Formulário.

Solicitando fotos Impressas:
Preencha o formulário com o seu endereço completo e anexe o comprovante da sua doação. Faremos o envio em até 2 dias úteis. O prazo médio para a entrega é de 5 a 10 dias úteis. Clique Aqui para acessar o Formulário.

Opções de Fotos:


  • Wallpapers para Computador
  • Wallpapers para Celular
  • Fotos Impressas
